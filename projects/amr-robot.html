Autonomous Mobile Robot

An end-to-end autonomous mobile robot platform integrating ROS 2, SLAM, navigation, sensor fusion, and embedded edge compute for real-time intelligent movement in dynamic environments.

Robotics Navigation Computer Vision Edge AI

Tools Used

ROS 2 Python C++ LiDAR Camera Raspberry Pi ESP32 micro-ROS SLAM

Project Overview

This project delivers a fully integrated autonomous mobile robot capable of mapping indoor environments, localizing itself, planning paths, and navigating safely using fused sensor data. The system combines high-level robotics software with embedded controllers and real-time perception pipelines.

The Problem

Building a mobile robot that can operate autonomously requires more than individual sensors or algorithms. The challenge was to unify mapping, localization, navigation, and low-level motor control into one reliable stack that runs on constrained edge hardware while remaining extensible for future AI features.

The Solution

I architected a ROS 2-based robotics stack with modular nodes for perception, planning, and control. LiDAR and camera data feed into SLAM and navigation pipelines, while ESP32 microcontrollers handle real-time motor control through micro-ROS. The Raspberry Pi serves as the main compute unit, coordinating autonomous behavior across the full system.

Workflow / Technical Process

  • Define robot hardware architecture and sensor layout
  • Implement ROS 2 nodes for LiDAR, camera, and odometry
  • Build and tune SLAM for environment mapping
  • Configure navigation stack for path planning and obstacle avoidance
  • Integrate ESP32 motor control via micro-ROS
  • Validate autonomous navigation in real-world test runs

Key Features

  • Real-time SLAM and localization
  • Autonomous path planning and navigation
  • Multi-sensor fusion with LiDAR and camera
  • Embedded motor control on ESP32
  • Modular ROS 2 architecture for future AI modules

Results / Impact

The robot successfully maps environments, plans routes, and navigates autonomously with integrated sensor feedback. The modular design supports adding computer vision and edge AI capabilities without rebuilding the core stack, making it a scalable foundation for intelligent robotics applications.

Skills Demonstrated

ROS 2 system design, SLAM, autonomous navigation, sensor fusion, embedded robotics, micro-ROS integration, C++ and Python development, and real-time intelligent systems engineering.
