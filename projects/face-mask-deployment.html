Face Mask Detection System

A real-time computer vision system that detects face mask compliance using deep learning, with a deployment-ready pipeline for live video inference and practical monitoring use cases.

Computer Vision Deep Learning Deployment

Tools Used

Python OpenCV PyTorch / TensorFlow YOLO ONNX Webcam Pipeline

Project Overview

This project builds a complete face mask detection pipeline from model training and evaluation through to real-time inference on live camera feeds. The system identifies whether individuals are wearing masks correctly and presents results in a format suitable for monitoring and deployment scenarios.

The Problem

Manual mask compliance monitoring does not scale and is prone to inconsistency. The challenge was to build an automated vision system that runs reliably on live video, handles varying lighting and angles, and can be packaged for deployment outside of a notebook environment.

The Solution

I developed a deep learning detection model trained on labeled face mask data and integrated it into a real-time OpenCV inference pipeline. The system captures webcam frames, runs detection, draws bounding boxes with classification labels, and supports export paths for optimized deployment formats.

Workflow / Technical Process

  • Collect and preprocess labeled face mask dataset
  • Train and evaluate deep learning detection model
  • Integrate model into OpenCV webcam capture pipeline
  • Optimize inference for real-time performance
  • Structure project for deployment-ready packaging
  • Validate detection accuracy across varied conditions

Key Features

  • Real-time face mask classification on live video
  • Deep learning object detection pipeline
  • OpenCV-based capture and visualization
  • Deployment-oriented project structure
  • Configurable inference for different hardware targets

Results / Impact

The system delivers reliable real-time mask detection suitable for access control, workplace monitoring, or public health scenarios. The end-to-end pipeline demonstrates the full path from model development to deployable computer vision software.

Skills Demonstrated

Deep learning model training, computer vision pipeline development, real-time inference, OpenCV integration, model deployment planning, and production-oriented AI software engineering.
