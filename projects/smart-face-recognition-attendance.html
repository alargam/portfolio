Face Recognition Attendance System

A real-time face recognition attendance system that automates check-in using OpenCV, LBPH face recognition, and a GUI-driven workflow for enrollment, identification, and attendance logging.

Face Recognition Attendance Automation OpenCV

Tools Used

Python OpenCV LBPH Face Recognizer Haar Cascades SQLite / CSV Logging GUI Framework

Project Overview

This project replaces manual attendance tracking with an automated face recognition pipeline. Users are enrolled with captured face samples, and the system identifies them in real time through a webcam, logging attendance automatically with timestamps and a user-friendly interface.

The Problem

Manual attendance systems are slow, error-prone, and difficult to scale. Organizations need a reliable way to verify identity and record attendance without cards, passwords, or manual data entry at the point of entry.

The Solution

I built a complete face recognition pipeline using OpenCV's Haar Cascade for face detection and LBPH for recognition. The system includes user enrollment, real-time webcam identification, automated attendance logging, and a GUI that makes the workflow accessible for daily use in classrooms or workplace environments.

Workflow / Technical Process

  • Capture and store labeled face samples for each user
  • Train LBPH face recognizer on enrolled dataset
  • Detect faces in real time using Haar Cascade classifier
  • Match detected faces against trained model
  • Log attendance with timestamp on successful recognition
  • Present results through a GUI for operators and administrators

Key Features

  • Real-time face detection and recognition
  • User enrollment and model training workflow
  • Automated attendance logging with timestamps
  • GUI for operation and administration
  • Lightweight pipeline suitable for local deployment

Results / Impact

The system automates attendance tracking with minimal manual intervention, reducing administrative overhead and improving record accuracy. It demonstrates a practical intelligent automation solution built with classical computer vision techniques and real-world usability in mind.

Skills Demonstrated

Face recognition pipeline development, OpenCV, real-time video processing, GUI application design, attendance automation, and practical intelligent system integration.
